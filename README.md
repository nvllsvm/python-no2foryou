# no2foryou
Replaces all `python2*` and `pip2*` commands with error messages.

Why? Because it's 2020 - stop using Python 2!

![no2foryou](pic.jpg)

## Installation
**Requires Python 3 (duh)**

```
$ pip install no2foryou
```

## Usage
```
$ python2
error: this version of python has expired. shame on you!

$ pip2
error: this version of python has expired. shame on you!
```
